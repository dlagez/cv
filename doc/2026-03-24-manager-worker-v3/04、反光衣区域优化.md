# 反光衣区域优化

## 1. 问题描述

当前反光衣 ROI 存在一个明显问题：检测框整体偏下，容易出现以下现象：

- 上半部分反光衣没有完整覆盖，胸前主体区域被截掉
- 下半部分吃进裤子、腿部、车辆或背景
- 在远景、侧身、骑车场景下，ROI 更容易只落在衣服下半段
- 颜色统计会被非反光衣区域稀释，导致 `vest_color` 不稳定

从当前效果看，问题核心不是“框太窄”单独导致，而是“上边界偏低 + 宽度参考不够稳定”共同造成的。

## 2. 优化目标

本次优化目标是：

- 把反光衣 ROI 整体上移，使其更贴近上半身背心主体
- 让 ROI 的上下边界更贴近人体结构，而不是依赖偏保守的框内比例
- 横向宽度优先参考双肩宽度，并适度外扩，避免只截到背心中间窄条
- 在不吃进头盔和脸部的前提下，尽量完整覆盖胸前和腹部反光衣主体

## 3. 推荐主方案

### 3.1 上边界：使用双肩高度

反光衣通常从肩线下方开始，因此 ROI 上边界应以双肩点为主，而不是从躯干中部开始。

建议：

- 使用 `5`、`6` 两个肩点作为上边界参考
- 取双肩较高位置作为基础肩线
- 在肩线基础上只做一个很小的向下偏移，避免吃进脖子和脸

建议公式：

```text
shoulder_y = min(left_shoulder_y, right_shoulder_y)
torso_y1 = shoulder_y + top_offset
```

其中：

- `top_offset` 建议为 `0.02 * person_h ~ 0.05 * person_h`
- 若画面中人物较远，可使用 `max(2px, 0.03 * person_h)` 这种下限保护

设计意图：

- 上边界尽量贴近反光衣肩部区域
- 只轻微下移，避免上边界过低导致 ROI 整体靠下

### 3.2 下边界：使用 11、12 点作为下沿参考

反光衣主体通常覆盖到髋部附近，因此下边界建议直接以 `11`、`12` 点作为下沿参考，而不是用过保守的固定比例。

建议：

- 使用 `11`、`12` 两个髋点作为 ROI 下边界参考
- 取双髋较低位置作为下沿基础
- 允许下边界略微再向下扩一点，用于容纳背心下摆

建议公式：

```text
hip_y = max(left_hip_y, right_hip_y)
torso_y2 = hip_y + bottom_offset
```

其中：

- `bottom_offset` 建议为 `0.03 * person_h ~ 0.08 * person_h`
- 但必须限制不能明显进入大腿区域

设计意图：

- 下边界和人体结构对齐
- 保证背心主体完整，不只取到上半段

### 3.3 宽度：使用双肩宽度并适度外扩

横向宽度建议优先使用双肩宽度，而不是仅依赖肩髋四点的最小包围结果。原因是反光衣主体通常以肩宽为更稳定的横向参考。

建议：

- 用双肩中点作为 ROI 横向中心
- 用双肩距离作为主宽度参考
- 在双肩宽度基础上左右各外扩一小部分，保证覆盖背心侧边

建议公式：

```text
shoulder_center_x = (left_shoulder_x + right_shoulder_x) / 2
shoulder_span = abs(right_shoulder_x - left_shoulder_x)
torso_half_w = 0.55 * shoulder_span ~ 0.70 * shoulder_span
torso_x1 = shoulder_center_x - torso_half_w
torso_x2 = shoulder_center_x + torso_half_w
```

等价理解：

- 总宽度约为 `1.10 ~ 1.40 * shoulder_span`

建议初始值：

- 总宽度先取 `1.20 * shoulder_span`

设计意图：

- 让 ROI 覆盖反光衣左右边缘
- 避免只取到中间一条窄区域
- 相比直接依赖髋点横向位置，双肩宽度在夜间和侧身场景下通常更稳

## 4. 推荐几何规则

当 `5、6、11、12` 四个点都可用时，建议采用以下主逻辑：

```text
shoulder_y = min(y5, y6)
hip_y = max(y11, y12)
shoulder_center_x = (x5 + x6) / 2
shoulder_span = abs(x6 - x5)

torso_x1 = shoulder_center_x - 0.60 * shoulder_span
torso_x2 = shoulder_center_x + 0.60 * shoulder_span
torso_y1 = shoulder_y + 0.03 * person_h
torso_y2 = hip_y + 0.05 * person_h
```

再增加边界约束：

- `torso_y1` 不得高于肩线太多，避免吃进头部
- `torso_y2` 不得低于人体框下部过多，避免吃进腿部
- `torso_x1 / torso_x2` 需要限制在人体框合理范围内

## 5. 回退逻辑

### 5.1 仅有双肩，无双髋

当 `11、12` 不稳定时：

- 上边界仍用双肩高度
- 下边界按肩宽和人体框高度估算

建议：

```text
torso_y1 = shoulder_y + 0.03 * person_h
torso_y2 = torso_y1 + max(1.05 * shoulder_span, 0.26 * person_h)
```

这样可以保证 ROI 仍然从上半身开始，而不是重新掉回偏下的比例框。

### 5.2 双肩、单髋可用

当只有一个髋点稳定时：

- 下边界优先使用可用髋点
- 再结合人体框比例做保守补偿

建议：

- 如果 `left_hip` 或 `right_hip` 仅有一个可用，则以该点作为下沿基础
- 再额外加上 `0.03 * person_h ~ 0.05 * person_h` 的补偿

### 5.3 ROI 过小

当按上述规则得到的 ROI 宽度或面积仍然过小时：

- 允许优先按双肩宽度继续补宽
- 再考虑回退到保守的人框比例方案

补宽优先级应高于直接回退，因为当前主要问题就是 ROI 偏窄、偏下。

## 6. 参数建议

建议增加或重点调整以下参数：

- `vest_top_offset_ratio`
  - 上边界相对肩线的向下偏移比例
- `vest_bottom_offset_ratio`
  - 下边界相对髋线的向下扩展比例
- `vest_shoulder_width_expand_ratio`
  - 基于双肩宽度的横向扩展比例
- `vest_min_width`
  - 反光衣 ROI 最小宽度
- `vest_min_height`
  - 反光衣 ROI 最小高度
- `vest_min_area`
  - 反光衣 ROI 最小面积

建议初始值：

```text
vest_top_offset_ratio = 0.03
vest_bottom_offset_ratio = 0.05
vest_shoulder_width_expand_ratio = 1.20
```

## 7. 预期收益

按该方案调整后，预期会带来以下改进：

- 反光衣 ROI 整体上移，更贴近背心主体
- ROI 横向范围更稳定，不容易只截到中间一条
- 颜色统计会更多来自胸前和腹部主色区域
- 减少裤子、腿部、地面和车辆进入 ROI
- 对黄绿色反光衣的识别会更稳定

## 8. 实施建议

建议按以下顺序落地：

1. 先将上边界改为“基于双肩高度 + 小幅下移”
2. 再将下边界改为“基于双髋高度 + 小幅下扩”
3. 再将横向宽度改为“基于双肩宽度 × 扩展系数”
4. 最后增加最小宽度/面积约束和回退逻辑

这样可以更容易观察每一步调整对 ROI 位置的影响。

## 9. 结论

本次优化建议的核心是：

- 上边界用双肩高度
- 下边界用 `11、12` 髋点高度
- 宽度以双肩宽度为主并适度拓宽

这比当前“ROI 偏下、偏窄”的方式更符合反光衣真实分布，也更利于后续做稳定的颜色识别。
